## What I Learned

Today I learned that every computer follows the Input → Processing → Storage → Output model.

📖 Chapter 1
1. What is a Computer?

Definition

A computer is an electronic device that accepts data as input, processes it according to a set of instructions, stores data if needed, and produces output.

Key Points
It works electronically.
It follows instructions (called programs).
It processes data.
It can store data.
It provides results as output.

2. What is Input?

Definition

Input is the data or instructions that are provided to the computer.

The computer cannot work unless it receives some form of input.

Common Input Devices
Device	Purpose
Keyboard	Enter text and commands
Mouse	Point, click, and select
Microphone	Capture voice
Webcam	Capture images and video
Scanner	Scan documents

3. What is Processing?
Definition

Processing is the stage where the CPU executes instructions and performs operations on the input.

The CPU:

Reads instructions.
Performs calculations.
Makes comparisons.
Produces results.

4. What is Storage?
Definition

Storage is where data and programs are saved.

There are two types:

Temporary Storage
RAM
Data is lost when the computer is turned off.
Stores running programs and active data.
Permanent Storage
SSD
HDD
Data remains even after the computer is turned off.

5. What is Output?
Definition

Output is the final result produced by the computer after processing.

Common Output Devices
Device	Purpose
Monitor	Displays text, images, and videos
Printer	Prints documents
Speakers	Play sound
Headphones	Audio output

7. What is Software?
Definition

Software is a collection of programs and instructions that tell the hardware what to do.

Software cannot work without hardware, and hardware is not useful without software.

Types of Software
System Software

Controls and manages the computer.

Examples:

Windows
Linux
macOS
Application Software

Used to perform specific tasks.
