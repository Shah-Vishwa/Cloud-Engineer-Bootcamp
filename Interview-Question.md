Interview Questions
Beginner
1. What is a computer?

Answer:

A computer is an electronic device that accepts input, processes data according to instructions, stores data when needed, and produces output.

2. What are the four basic functions of a computer?

Answer:

Input
Processing
Storage
Output
3. What is hardware?

Answer:

Hardware consists of the physical components of a computer, such as the CPU, RAM, motherboard, keyboard, and monitor.

4. What is software?

Answer:

Software is a collection of programs and instructions that tell the hardware what to do.

5. What is the difference between hardware and software?

Answer:

Hardware refers to the physical parts of a computer, while software refers to the programs and instructions that run on the hardware.
